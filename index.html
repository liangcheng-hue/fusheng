项目核心摘要

肤生 Skinborn —— 原生肤感香氛品牌。像皮肤自己透出来的原生体香，半米外像没喷，贴身闻清润自然。
维度结论
赛道选择切入200–300元「原生肤香」垂直赛道。50–150元平价伪体香个护已是红海；200–400元区间尚无品牌独占"无香水感原生体香"心智。
品牌定位品类占位型。心智占位：像皮肤自己透出来的原生体香。英文名 Skinborn（肌肤所生）。品牌级 slogan：肤香天生，无需刻意。
首发产品三款滚珠香氛油，共享统一贴肤香基，全系列达标"伪体香金标准"。正装 200–300 元，试香装 9.9 元。
目标用户20–26 岁一二线城市精致女性，月可支配收入 3000–8000 元。购买动机：自我愉悦 ＞ 社交细节加分 ＞ 异性好感。
冷启动小红书为核心阵地。"约会伪体香"承接搜索流量 → 逐步传递"原生肤香"品牌内核。内容比例：70% 场景种草 + 30% 品牌心智。
启动预算<10 万。调香打样 + 首批大货优先，包材占比控制在 15% 以内。
创始人优势小红书香水博主（~1万粉丝），内容能力极强（周更3篇零成本），供应链通路（朋友品牌→工厂）。

品类占位型 —— "原生肤感香氛"品类没有领导品牌。第一个站出来的品牌定义这个品类。

生态位公式

原生体香需求 ∩ 伪体香搜索流量 ∩ 香水博主审美判断力

首发产品体系 · 统一伪体香标准

全系列共享统一贴肤香基，三款均适配"伪体香"营销逻辑，覆盖不同约会场景。产品形态：滚珠香氛油。正装 200–300 元，试香装 9.9 元。

一、全系列统一伪体香标准（品牌底线，三款 100% 达标）

标准具体要求
① 无冲击感低酒精配方，上身无刺鼻冲感。10 分钟内完全与角质层融合，无"刚喷完香水"的突兀边界。
② 晕染式散香香味呈晕染状附着在皮肤表层。像捧花一天染上的气息、逛森林沾到的露水感——非喷洒式集中香源。
③ 留香尺度贴身有效留香 6–8 小时。30cm 近距离闻清润不齁，1m 距离悠悠可闻，社交距离不刻意不扰人。
④ 伪体香金标准盲测场景下，多数人无法精准分辨是香水还是原生体香。男性高频反馈「你本身就很好闻」——而非「你香水很好闻」。

二、首发三款香型详情

香型一 · ________

核心场景：刚沐浴完的清冷少女肤感

画面锚点：浴室水汽未散，栀子花香混着洁净肤感，易碎又有韧劲的疏离感。

香调层次（无突兀三调，全程平缓融合）

顶层气息
极淡蒸馏水汽 + 栀子花瓣露感。仅上身 10 分钟内可捕捉清冽感。
中层融合
柔润洁净皂感 + 薄透栀子花香。如水汽裹着花香渗进皮肤，无甜腻感。
基底肤香
高纯度白麝香 + 柔肤内酯。与肌肤完全贴合，最终留存的原生体香底色。
伪体香营销锚点：初次约会清冷氛围感神器。凑近闻像刚洗完澡的干净体香。男生只会觉得你天生自带洁净感——完全想不到是喷了香氛。

香型二 · ________

核心场景：带绿意的清甜少女肤感

画面锚点：穿绿纱裙跑过果园草地，青苹果脆甜混着草叶露水，沾在皮肤上的鲜活气息。

香调层次（无突兀三调，全程平缓融合）

顶层气息
极淡青苹果脆感 + 嫩草叶尖绿意。酸大于甜，无果调香水的齁甜感。
中层融合
柔和青草水汽 + 淡苹果皮清香。像风裹着果香扫过皮肤——"染上"而非"喷上"。
基底肤香
统一白麝香汤底。果绿意锁在肤感里，甜而不飘，鲜活不跳脱。
伪体香营销锚点：户外约会 / 夏日专属。男生会觉得你自带清甜少女感——像刚吃完苹果、逛完公园的自然气息，完全不刻意。

香型三 · ________

核心场景：晨起被窝里的软嫩少女肤感

画面锚点：阳光晒过的棉被里，刚睡醒的女生，皮肤带着软乎乎的奶绒感，干净又娇嫩。

香调层次（无突兀三调，全程平缓融合）

顶层气息
极淡阳光暖感 + 棉麻纤维气息。像晒过太阳的被子蹭在皮肤上的温度。
中层融合
婴儿般淡奶感 + 极柔花香底蕴。奶味淡到几乎无，仅留存软嫩肤感联想，无齁感。
基底肤香
暖调白麝香 + 柔肤内酯。暖而不闷，软而不腻——皮肤本身透出来的软嫩质感。
伪体香营销锚点：近距离亲密接触绝杀。抱起来是软乎乎的干净奶感。男生会以为你天生皮肤就带着甜香——毫无人工痕迹。

三、产品基础设定

维度设定
核心形态滚珠香氛油（贴肤融合度最高，最适配原生体香体验）
定价策略正装 200–300 元区间；配套 9.9 元试香装降低试错成本

50–150 元平价伪体香个护是红海；200–400 元"无香水感原生肤香"垂直赛道是心智蓝海。

三层竞品结构

层级品牌 / 产品核心问题

核心用户画像

目标用户：20–26 岁，一二线城市精致女性。月可支配收入 3000–8000 元。购买动机排序：自我愉悦 ＞ 社交细节加分 ＞ 异性好感。
维度画像详情
年龄与城市20–26 岁，一二线城市（北上广深 + 新一线）
月可支配收入3000–8000 元。200–300 元香氛 = 轻奢犒赏级别——不需要犹豫的价格，但有品质感知
消费行为小红书种草→线上购买。64% 偏爱小样试用。对"千人千面"的伪体香概念有认知但多次购买失败。
购买动机排序① 自我愉悦（"喷了觉得自己又松弛又美"）＞ ② 社交细节加分（"约会中隐隐约约的好闻"）＞ ③ 异性好感（"他闻到了——但觉得是我自己的味道"）
核心痛点不满市面伪体香的香水感、化工感——买了"伪体香"回来，发现还是有"喷了香水"的边界。排斥刻意浓香，追求"天生好闻"的体香质感。
关键场景约会前最后一步、日常出门前、洗澡后。决策模式：穿搭搞定、妆容搞定——喷香不想再纠结。

核心洞察：女生的"试错困境"

约会前——衣服不能每件都试、口红不能每支都涂。她已经在穿搭和妆容上消耗了决策精力。喷香这件事，她不想再纠结。她需要一个"闭眼喷就对了"的默认选项。

肤生解决的：在所有需要做决定的时刻里，至少有一件事是确定的。

运营冷启动策略

小红书为核心阵地。"约会伪体香"承接搜索流量 → 逐步传递"原生肤香"品牌内核。内容比例：70% 场景化种草 + 30% 品牌心智内容。

内容策略

内容类型占比目的选题方向
场景化种草70%抓搜索流量、促首次购买"约会伪体香推荐""第一次约会怎么喷""男朋友说你是不是有体香"——承接小红书搜索量最大的约会香关键词
品牌心智内容30%建品牌认知、促复购与推荐"什么是真正的原生肤香""为什么市面伪体香都有香水感""滚珠香氛油和香水的区别"——建立品类教育壁垒

信任螺旋 6 层内容节奏

层级时间内容方向
① 创始人故事0–1 月"我是香水博主。看你们买回来的伪体香都不对。所以我来做。"
② 态度内容1–2 月"市面伪体香的香水感问题——一支一支拆给你看。"
③ 场景种草2–3 月"约会前不用想的香——是什么样的。"
④ 方案呈现3–4 月"肤生三款。匹配不同约会场景。这就是原生肤香。"
⑤ 用户 UGC4–6 月"她们用了之后——让男朋友自己说。"
⑥ 品牌心智6 月+"肤生 = 原生肤香标准。"

渠道规划

阶段渠道动作
冷启动期（0–6 月）小红书 + 私域创始人账号 = 品牌主阵地。私域沉淀首批 100 个种子用户，收集约会反馈→产品迭代。
验证跑通后天猫 + 抖音小红书验证产品力→天猫承接品牌搜索流量→抖音内容扩圈。

创始人 IP 优势

小红书香水博主（~1万粉丝），周更 3 篇零成本。内容能力 = 冷启动最大引擎。品牌内容不单独开号——从个人号自然过渡，粉丝信任→品牌信任无缝衔接。

⚠️ 不做什么

不做大型事件营销（预算约束）。不做付费投放（品牌初期聚焦产品 + 内容）。不在冷启动期分散到多平台（小红书一个阵地打透再扩）。

预算与风险预案

启动预算 <10 万。包材占比控制在 15% 以内，优先保障调香品质与首批大货。内容推广成本 ≈ 0。

首年预算结构

板块占比估算说明
调香打样 + 配方微调30%~3 万白麝香基底改良 + 三款变奏方向 + 多轮打样。优先保障。
首批大货（100–300 支）25%~2.5 万灌装 + 质检 + 基础包装。滚珠瓶公模优先，不开私模。
包装设计 + 打样15%~1.5 万外盒设计 + 瓶身定制涂层 + 内衬。成本上限 15%，优先保障调香。
化妆品备案 + 检测10%~1 万化妆品备案、微生物检测、安全性评价。合规为底线，不可跳过。
平台扣点 + 物流售后8%~0.8 万小红书店铺扣点、快递、售后物料。
基础推广 + 试香装7%~0.7 万9.9 元试香装制作。少量小红书信息流测试（可选）。
机动预留5%~0.5 万不可预见支出。
内容生产0%0创始人自产内容，零成本。

风险应对预案

风险概率影响应对
配方改不到位——30 分钟后跑偏，不像体香中高延迟上线，不妥协产品及格线。换调香师 / 基底，以盲测通过为准。
竞品快速跟进——冰希黎等品牌推"原生肤香"产品线高中不恋战速度，比拼品质。"伪体香金标准"= 竞品短期无法复制的产品力壁垒。靠品类心智 + 香型矩阵应对。
供应链品质不稳——不同批次味道不一致低中高建立标准样本封存制度，每批灌装前对照。同时储备 1–2 家备选工厂。
化妆品备案延迟——合规流程超预期低中提前启动备案流程，预留 2–3 个月周期。产品配方冻结后第一时间送检。
首批销售不达预期中中回到联名款路径——和朋友品牌合作出联名款降验证成本。Plan B：联名验证→独立品牌延后。

P0 · 本周就能做的 3 件事

① 发"我决定做一个品牌"的第一篇小红书。真实、不加修饰——"每次看你们买回来的伪体香都不对。所以我来做。叫肤生。"

② 联系朋友品牌老板。"大哥，我想做滚珠香氛油，方向想清楚了。工厂资源能介绍不？"

③ 在下一篇约会香推荐里埋梗。"最近在想——要不要自己做一支。叫肤生。你们觉得呢？"——测用户反应。
